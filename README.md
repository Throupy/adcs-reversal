# AD CS Reversal
This repo contains some pseudo-reversal of AD CS installation components. Mainly done in support of some blog posts

# Disclaimer
This repository contains annotated pseudocode reconstructed through static analysis of windows system components. It was produced for security research and interoperability analysis purposes.
The pseudocode in this repository is not extracted source code. It is a analytical reconstruction produced using a disassembler and decompiler (Ghidra), with variables renamed, control flow restructured, and commentary added by the author (me). It represents the author's understanding of the binary's behaviour, not a copy of any original source.
This research was conducted to understand internals of AD CS role configuration processes.
No cryptographic keys, credentials, or other sensitive material are present in this repository. This research does not facilitate circumvention of any security mechanism — it documents existing behaviour of a Windows role configuration tool.
